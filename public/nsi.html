Les Types


Hello world, c'est quoi ? C'est simplement un texte, qu'on appellera en programmation string ou encore chaîne de caractère (array of chars). Les strings sont encerclés par des guillemets, il y a différentes méthodes pour mettre vos guillements, vous pouvez en mettre un seul ('), mais vous remarquerez qu'avec un seul guillement ça marche pas quand vous tentez de faire un print('J'aime les glaçons'), c'est très simple, le programme est confus, il pense que votre string ne contient qu'un J, le reste, il sait pas ce que c'est du coup il renvoie des erreurs. Dites vous que votre compilateur est comme une personne psychorigide qui se tire une balle dès que vous faites une faute d'ortographe, ici, c'est pas une faute d'orthographe mais une faute de syntaxe. Solution, mettre des doubles guillements ("). Et si jamais vous voulez mettre des doubles-guillements dans votre string, vous aurez le même problème, ici la solution sera de mettre des triples-guillements ('''). Il est possible de concaténer des strings pour former des phrases ou des mots par exemple. L'output est la sortie du programme, ce qu'il renvoit dans la console. # permet de faire des annotations, tout bout de code précédé par ce caractère sera ignoré durant l'éxecution, cela peut s'avérer très pratique, entre autres pour prendre des notes. Exemple concret : Après les strings, y a les nombres. Les nombres, c'est un peu différent parce qu'on ne met pas de guillements. On distingue les int (integer, de l'anglais, se traduisant par entier), c'est, basiquement tous les nombres relatifs.

Puis, il y a les float, c'est tous les autres nombres, même si ça semble évident je vais le préciser ; int ⊂ float et la réciproque est fausse. Un string peut contenir des chiffres, mais va les gérer différement (vous allez comprendre de quoi je parle quand on passera aux opérateurs), il est impossible de concaténer un string et un float/int. Il est possible d'effectuer des opérations avec les float/int. En convertissant un int en float, il va se taper un .0, et à l'inverse, en convertissant un float en int, ça supprimera simplement ses décimales pour faire un nombre entier, sans l'arrondir, c'est pour ça que 1.7 se transforme en 1 et non en 2. Si vous voulez arrondir, il y a une fonction pour nommée round(). Je vous laisse tester tout cela par vous-même. Il reste deux types que je laisserai pour plus tard, les listes et les dictionnaires (simplement parce que je n'ai pas encore abordé la notion de variable et cette dernière est indispensable pour comprendre les listes et les dictionnaires).

Les variables

Vous pouvez faire un truc totalement génial, c'est stocker un int, un float ou encore un string dans une variable (et bien d'autres mais c'est pour plus tard) dans une boîte. Les variables sont indispensables pour programmer, on ne peut pas nous en passer. Plutôt que d'écrire 5 lignes que vous lirez à moitié endormis sur votre lit je vais vous montrer un exemple : On peut donner n'importe quel nom à notre variable du moment qu'il ne commence pas par un chiffre et qu'il ne contient pas d'accent / caractère non-ascii (genre ℜ ou ⊗)
Note : Il est possible de nommer des variables ainsi dans les dernières versions de Python.
Il est possible de modifier le contenu de la boîte, le supprimer, le remplacer, l'incrémenter, le décrémenter... bref t'as compris c'est génial les variables.
