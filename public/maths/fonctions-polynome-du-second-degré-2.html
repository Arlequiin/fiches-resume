Forme canonique d'un polynôme du second degré

(Cliquer sur ce titre pour vous tester)
Propriété : Toute fonction polynôme du second degré ƒ définie sur ℝ par ƒ(𝑥) = a𝑥² + b𝑥 + c s'écrit sous la forme ƒ(𝑥) = a(𝑥-α)²+β

Démonstration
Puisque a ≠ 0 :
$$a(x^{2}+\frac{b}{a}x) + c = a((x+\frac{b}{2a})^{2}-\frac{b^{2}}{4a^{2}})+c\\f(x) = a(x+\frac{b}{2a})^{2} - \frac{b^{2}}{2a})^{2}\\\text{En notant }\Delta = b^{2} - 4ac\text{, on a: }\\f(x) = a(x+\frac{b}{2a})^{2}-\frac{\Delta}{4a}\\\text{En posant }\\\alpha = -\frac{b}{2a}\text{ et }\beta = \frac{\Delta}{4a}\\\text{on obtient : }f(x) = a(x-\alpha)^{2}+\beta$$

Résolution de l'équation a𝑥²+b𝑥+c=0


Définition : Si ƒ est définie ℝ par ƒ(𝑥) = a𝑥²+b𝑥+c, avec a ≠ 0, on appelle discriminantde ƒ le nombre : Δ = b² - 4ac
Propriété : Résolution de l'équation a𝑥²+b𝑥+c=0
On calcule le discriminant : Δ = b² - 4ac
  • Si Δ < 0 : L'équation n'admet aucune solution dans ℝ
  • Si Δ = 0 : l'équation admet une unique solution

    • $$\text{S }= \{-b/2a\}$$

  • Si Δ > 0 : L'équation admet deux solutions ;

    • $$\text{Solution 1 } = \frac{(-b - \sqrt{\Delta})}{2a} \\\text{Solution 2 } = \frac{(-b + \sqrt{\Delta})}{2a} $$


    Propriété : Factorisation de ƒ(𝑥) = a𝑥² + b𝑥 + c suivant le signe de Δ
    • Si Δ < 0 : Pas de factorisation possible
    • Si Δ = 0 : ƒ(𝑥) = a(𝑥-𝑥₀)², ou 𝑥₀ est l'unique racine de ƒ
    • Si Δ > 0 : ƒ(𝑥) = a(𝑥-𝑥₁)(𝑥-𝑥₂), où 𝑥₁ et 𝑥₂ sont racines de ƒ


    And that's pretty much it
